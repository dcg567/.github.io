Dropify iOS Application

In the development of the Dropify iOS app, I utilised Swift as the primary programming language, leveraging its powerful features alongside the SwiftUI framework for a smooth user interface.

The app integrates Firebase for user authentication, allowing seamless sign-in, sign-up, and sign-out functionalities.

The project involved creating a custom tab bar for easy navigation, implementing a search bar for quick item discovery, and enabling notifications to alert users about upcoming sneaker drops.

Through diligent coding and design, I was able to create an aesthetically pleasing and fully functional application that operates on iPhones running iOS 16 or higher.

My app is a user-friendly platform that enhances the sneaker and streetwear shopping experience, ensuring collectors never miss out on their desired items.
