ANTLR4 Compiler Grammar Implementation

In this project, I implemented a grammar for an ANTLR4-based compiler, focusing on parsing integers, statements, and expressions.

I began by familiarizing myself with ANTLR4 through Zoom tutorials and built my grammar file, referencing the provided materials.

To address challenges, such as the representation of integers and left recursion issues, I modified rules and introduced new ones like primaryExp.

I also tackled reserved words and syntax errors while ensuring correct parsing through terminal tests using grun.

The driver code was adapted from examples to validate syntax errors and confirm successful parsing.

I ran the program via the command line, successfully parsing multiple test cases.

Technologies used included ANTLR4 and Java.
